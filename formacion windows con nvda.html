Formación básica de NVDA y Windows

Se les invita a las personas con una discapacidad visual ya sean personas ciegas o de baja visión a tomar este curso, en donde comenzaremos desde lo más básico en el uso de NVDA hasta lograr dominar el uso de NVDA y poder trabajar con el sistema operativo Windows de forma autónoma y la navegación web.

Este curso les preparará para que puedan sacar la certificación de NVDA Expert si así lo desean.

Requisitos para tomar este curso

El curso se impartiría de forma virtual así que tomando esto en cuenta a continuación se detalla:

  • Tener computadora ya sea de escritorio o laptop con conexión a internet.
  • En principio tener la aplicación de Google Meet ya que las lecciones se impartirían por ese medio, aunque luego con la población estudiantil se puede negociar si se usa otra plataforma.
  • Para población que no sea de Costa Rica si se les pide tener la aplicación de PayPal, misma que se utiliza para transferencias internacionales.
  • Tener instalado en sus computadoras última versión de NVDA
  • De ser posible esto no es obligatorio saber usar el Whatsapp de escritorio, por temas de facilidad y también para poder entregar las tareas por dicho medio.

Duración

La duración sería de aproximadamente 3 meses, tomando en cuenta que las clases se impartirían 2 veces por semana con una duración de 2 horas por día.

¿Qué podrá hacer la población luego de tomar esta formación?

Al finalizar el curso, los estudiantes podrán usar NVDA para realizar una amplia gama de tareas en el ordenador, incluyendo:

  • Navegar por el sistema operativo Windows, incluyendo el menú Inicio, la barra de tareas y las ventanas.
  • Leer y editar texto en aplicaciones de procesamiento de textos, como wordpad.
  • Explorar páginas web y rellenar formularios.
  • Navegar por archivos y carpetas.
  • Usar herramientas de accesibilidad, como el lector de pantalla y el sintetizador de voz.
  • Saber utilizar los diferentes modos de navegación de NVDA.
  • Configurar y adaptar NVDA a sus necesidades personales.
  • Crear y trabajar con copias portables.
